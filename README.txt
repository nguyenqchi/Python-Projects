Flappy bird game
	
*Note: Pygame needs to be installed to run this program	

How to run the program

- Main game to play: 1. Game.py
- Classes to test: 
	Birdclass.py: create the Bird object and control the bird's movement
	Pipeclass.py: create Pipe object
	buttonClass.py: creates button
- Txt file to store the data of high scores recorded: Flappy Bird Leaders.txt
- img folder to store pictures to be used for game design
- sound folder to store the sounds used in the game
